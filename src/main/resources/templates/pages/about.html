About the Site

Balamb is a catalogue for my personal home library, and is being developed as a passion project on multiple fronts. Not only is it about collecting and showing off books, it is also driven by a love of curating and maintaining information. I am slowly working to get all my books added, as well as continuing to develop and improve the site itself. Balamb is developed using the Java Spring framework, Bootstrap, and the Thymeleaf templating engine. Expect further updates to include more search/filtering options, a redesigned layout, various invisible backend improvements, and support for more media types such as video games. The updates page includes details about new features, upcoming changes, and known bugs/issues.

In a future update this page will also include some statistics about the contents of the library. Check back soon!
